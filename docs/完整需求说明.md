# 完整需求说明

## 项目定位

开发一个 Windows 本地运行的“百度竞价日报/小时报自动化工具”。

当前阶段只做自动写 Excel，不做 QQ、不做截图。

## 最终目标

1. 打开百度营销后台；
2. 进入云南银康账户；
3. 读取三个账户当天指定时段的百度推广数据：展现、点击、消费；
4. 打开快商通后台；
5. 读取对应时段的对话转化数据：总对话、有效、有效转潜、总转潜；
6. 打开本地 Excel 文件；
7. 定位 `时段数据` sheet；
8. 根据当天日期、当前时段、账户名称、字段表头，把数据写入对应单元格；
9. 保存 Excel；
10. 输出日志和自检报告。

## 固定时段

- 11点
- 15点
- 18点

运行时可通过参数指定：

```cmd
python main.py --mode run --period 15点
```
