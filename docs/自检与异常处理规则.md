# 自检与异常处理规则

## Excel 结构自检

必须检查：

- Excel 文件是否存在；
- `时段数据` sheet 是否存在；
- 是否找到三个账户区域；
- 是否找到日期列；
- 是否找到时段列；
- 是否找到展现、点击、消费列；
- 是否找到总对话、有效、有效转潜、总转潜列；
- 是否没有误把汇总区域当成账户区域。

## 百度数据自检

必须检查：

- 是否读取到三个账户；
- 是否没有多账户；
- 展现、点击、消费是否为数字；
- 消费是否非负；
- 后台账户是否能映射到 Excel 标准账户。

## 快商通数据自检

必须检查：

- 总对话是否为数字；
- 有效是否小于等于总对话；
- 有效转潜是否小于等于有效；
- 总转潜是否小于等于总对话；
- 无法归属账户的对话是否进入异常名单。

## 写入后自检

必须检查：

- 写入单元格是否和目标账户一致；
- 写入值是否和后台读取值一致；
- 没有写入汇总区域；
- 没有改其他 sheet；
- Excel 已保存；
- 生成日志和报告。

## 必须中断的情况

| 异常 | 动作 |
|---|---|
| 找不到 Excel 文件 | 要求用户重新配置路径 |
| 找不到 `时段数据` sheet | 中断 |
| 找不到三个账户区域 | 中断并输出疑似标题 |
| 找不到日期 | 中断 |
| 找不到当前时段 | 中断 |
| 找不到字段 | 中断或跳过对应字段并报告 |
| 百度后台账户不足 3 个 | 中断 |
| 百度后台账户多于 3 个 | 中断 |
| 写入后复核不一致 | 中断并提示人工检查 |
