# v2026.7.19.106 更新说明

## 本版更新

- 在线更新改为先提示、再下载：检测到新版本后显示“更新”，不再启动即静默下载。
- 下载期间持续显示百分比；失败后保留重试入口，下载完成后显示“更新重启”。
- 固定从 `179068898-dotcom/Hourlyreport-Automation` 获取最新 Release，并补充上一版本识别新资源的回归测试。
- 精简未使用的 Qt 组件，在线更新包由约 55 MB 降至约 39 MB，保留 Windows、图片和 WebP 等实际运行能力。
- 调整 API / 浏览器模式按钮、默认小时报时段、日报标题和顶部菜单间距。
- EXE 使用黑底圆角图标；任务栏和托盘使用透明底放大小螃蟹图标。
- 发布构建测试使用隔离输出目录，避免测试过程覆盖正式 ZIP。

- 在线更新不会覆盖 `configs/`、`secrets/`、日志、报告、备份、商务通导出目录或浏览器数据。
- 104 的旧更新界面会在后台自动下载且隐藏失败状态；若同事电脑始终看不到 106 提示，请人工完整安装一次 106。升级到 106 后，后续版本使用新的可见下载流程。
