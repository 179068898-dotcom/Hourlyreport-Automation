# v2026.7.22.108 更新说明

## 本版重点

- GUI 小时报、日报标题右侧新增 `open` 独立开关。
- `open` 开启时任务完成后自动打开当前项目 Excel；关闭时只完成任务并保留日志，不自动打开 Excel。
- 新增顶部 `帮助` 菜单。
- `帮助 -> 关于小螃蟹` 增加 Codex 风格版本信息弹窗。
- `帮助 -> 检查版本` 支持手动触发 GitHub Release 更新检测。

## 使用说明

- 小时报和日报的 `open` 开关互不影响。
- 开关默认开启，保持旧版本“做完自动打开 Excel”的习惯。
- 任务运行中开关会暂时禁用，任务结束后恢复。
- 手动检查版本不会自动下载；检测到新版本后仍需点击顶部更新按钮。

## 发布资产

- 完整安装器：`Hourlyreport_automation_setup_v2026.7.22.108.exe`
- 在线更新包：`Hourlyreport_automation_v2026.7.22.108.zip`

## 验证重点

- GUI 可以正常启动。
- 小时报 `open` 关闭后，任务完成不自动打开 Excel。
- 日报 `open` 开启后，任务完成仍自动打开 Excel。
- 帮助菜单可以打开关于窗口。
- 手动检查版本可以触发现有在线更新检查流程。
