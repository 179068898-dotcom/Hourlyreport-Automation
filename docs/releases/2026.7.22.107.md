# v2026.7.22.107 更新说明

## 本版更新

- 修正快商通“有效对话”统计口径：`有效对话` 只包含 `有效-三句话` 和 `转潜-有效`；`有效-一般` 独立写入 `一般有效` / `一般有效对话`。
- 小时报字段同步为 `有效对话`、`一般有效`、`有效转潜`，其中 `转潜-有效` 会同时计入有效对话和有效转潜。
- 修复新电脑环境安装兼容性：不再优先使用系统 Python，改为下载并使用隔离的项目专用 Python 3.14.5 NuGet 运行时，避免本机 Python 3.14.6 影响安装。
- 新增 `requirements-runtime.lock.txt` 依赖锁定文件；安装时优先使用 lock，保证同事电脑运行依赖版本一致。
- 新增脱敏诊断包入口：`.venv\Scripts\python.exe main.py --mode diagnostic-bundle`，会收集配置、报告和日志摘要，但跳过 secrets 并脱敏密码、Token、密钥等敏感内容。
- 新增日志归档入口：`.venv\Scripts\python.exe main.py --mode archive-logs`，默认归档 14 天以前的 `.log` 文件到 `logs/archive/`。
- 新增百度授权云端集中刷新：桌面端优先从腾讯云 SCF / COS 获取已刷新的百度 Token；本地授权过期时由云端统一刷新，减少同事电脑重复导入授权文件。
- 修复在线更新提示隐藏问题：检查更新时按钮保持可见，检查失败显示“重试”，下载失败回到可重新点击的“更新”，避免 106 客户端因网络/API 请求失败而静默无提示。
- 发布包排除规则加强：`.lock` 本地锁文件不再进入在线更新包或完整安装器。

## 兼容说明

- 本次发布不覆盖 `configs/`、`secrets/`、日志、报告、备份、快商通导出目录或浏览器数据。
- 完整安装器仍不包含真实账号密码和 OAuth Token；真实授权继续通过独立 `.baidu-secrets` 配置包导入。
- 脱敏诊断包仅用于排障，不应替代真实授权配置包。
