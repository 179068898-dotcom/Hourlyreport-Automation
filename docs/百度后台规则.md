# 百度后台规则

## 读取范围

读取三个账户当天数据：

- 银康01
- 银康银屑02
- baidu-银康03

## 字段

- 展现
- 点击
- 消费

## 账户映射

|---|---|
| 银康01 | 银康01 |
| 银康银屑02 | 银康银屑02 |
| baidu-银康03 | 银康03 |

## 自动化原则

1. 优先使用 URL、文本、表头、表格结构、选择器；
2. 不依赖绝对坐标；
3. 如果未登录，停下来提示人工登录；
4. 登录态使用 `browser_profile/` 持久化；
5. 页面解析失败时输出 `reports/page_debug_screenshot.png` 和 `reports/page_debug.html`；
6. 读取到的数据只输出 JSON，不直接写 Excel，直到整合阶段。
