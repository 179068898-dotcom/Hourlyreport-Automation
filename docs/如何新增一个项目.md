# 如何新增一个项目

本工具支持多项目配置。每个项目一个 JSON 文件，统一放在：

```text
configs/projects/
```

## 操作步骤

1. 复制模板文件：

```text
configs/projects/project_template.json
```

2. 把复制出来的文件改名为项目 ID，例如：

```text
configs/projects/new_project.json
```

3. 修改 `project_id` 和 `project_name`：

```json
{
  "project_id": "new_project",
  "project_name": "新项目名称"
}
```

4. 修改 Excel 路径：

```json
"excel": {
  "path": "D:/新项目/2026竞价数据.xlsx",
  "hourly_sheet": "时段数据",
  "daily_sheet": "百度",
  "engine": "openpyxl"
}
```

5. 修改日报 sheet 和小时报 sheet：

- `excel.hourly_sheet`：小时报写入的 sheet；
- `excel.daily_sheet`：日报写入的 sheet。

6. 修改商务通导出目录：

```json
"kst": {
  "export_dir": "D:/新项目/商务通导出",
  "auto_pick_latest": true,
  "max_file_age_minutes": 30
}
```

7. 修改百度凭据 profile：

```json
"baidu": {
  "credential_profile": "new_project_baidu"
}
```

8. 修改账户映射。每个账户都要配置：

```json
{
  "standard_name": "账户01",
  "baidu_names": ["百度后台账户名"],
  "excel_name": "Excel里的账户区域名",
  "kst_ids": ["商务通推广ID"],
  "kst_names": ["商务通账户名或别名"]
}
```

9. 在 `secrets/secrets.json` 中添加对应的百度账号密码：

```json
{
  "baidu": {
    "new_project_baidu": {
      "username": "百度账号",
      "password": "百度密码"
    }
  }
}
```

10. 打开菜单，选择 `切换项目`，切换到新项目。

11. 运行 `检查运行环境`，确认当前项目配置、Excel、sheet、商务通导出目录和百度凭据是否可用。

12. 环境检查通过后，再运行日报或小时报。

## 注意事项

- `project_id` 必须和文件名一致，方便识别和切换。
- `excel.engine` 默认使用 `openpyxl`，适合 WPS 环境，不要求安装 Microsoft Excel。
- `secrets/secrets.json` 不会进入发布包，也不要发给别人。
- 如果 doctor 提示 Excel 路径不存在，先修正 `excel.path`。
- 如果 doctor 提示商务通导出目录不存在，先修正 `kst.export_dir`。
