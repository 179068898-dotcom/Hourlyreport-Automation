# Excel 识别与写入规则

## 最高优先级

1. 不允许重建 Excel 文件；
2. 不允许修改无关 sheet；
3. 不允许修改 `每日时段统计数据 / 汇总区域 / 截图区域 / 公式区域`；
4. 不允许写死固定坐标；
5. 写入前必须备份；
6. 写入后必须重新读取单元格复核。

## Sheet

目标 sheet：`时段数据`

## 区域

| 区域 | 处理方式 |
|---|---|
| 每日时段统计数据 / 汇总区域 | 只读，不写入 |
| 银康01 | 可写入 |
| 银康银屑02 | 可写入 |
| 银康03 | 可写入 |

## 识别步骤

1. 扫描 used range 的所有非空单元格；
2. 对单元格文本做标准化；
3. 输出 `reports/sheet_text_dump.csv`；
4. 根据账户别名识别账户标题；
5. 用账户标题之间的行号判断账户区域；
6. 在账户区域内部寻找字段表头；
7. 输出 `reports/excel_structure_report.json`。

## 文本标准化

- 去前后空格；
- 去换行；
- 去制表符；
- 去全角空格；
- 多个空格合并；
- 英文统一小写；
- 中文保持不变。

| 有效转潜 | 有效转潜 |
| 总转潜 | 总转潜 |

## 写入规则

写入时必须定位：

1. 标准账户；
2. 账户区域；
3. 当前日期行；
4. 当前时段行；
5. 字段列；
6. 目标单元格。

找不到任何一个环节都必须中断。
