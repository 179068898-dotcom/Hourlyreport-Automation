# 同事使用说明

内部版同步日期：2026-07-15

## 最简单的使用方式

1. 完整解压发布包，不要只把 EXE 单独拖出来。
2. 双击根目录中的 `hourlyreport_automation.exe`。
3. 第一次启动会自动检查环境；缺少 Python 时会显示进度并自动下载、安装项目专用 Python 与依赖。
4. 在“系统配置”中更新本机 Excel 路径和百度账号密码。
5. 选择项目、小时报时段或日报日期后执行。

GUI 成功完成小时报或日报后会自动打开当前项目 Excel。

## 窗口与托盘

- 控制台打开时，Windows 任务栏会显示程序小图标。
- 点击控制台右上角 `X` 只会隐藏窗口，程序、托盘和正在执行的任务不会退出。
- 左键单击任务栏通知区域的托盘图标，可以重新打开控制台。
- 真正退出程序只能使用托盘图标右键菜单中的“退出程序”，或控制台“系统配置”中的“退出程序”。

## 第一次升级到自动更新版

`2026.7.15.101` 之前安装的版本没有在线更新功能，无法自己发现新版本。需要管理员先发放一次 `101` 完整基线包：

1. 退出旧程序。
2. 保留本机的项目配置、账号密码和 Excel 文件。
3. 将 `101` 基线包完整解压并覆盖程序文件，不要只替换 EXE。
4. 启动 `hourlyreport_automation.exe`，以后新版本会在启动时自动检查。

看到标题栏出现更新按钮时，等待下载完成后点击“更新”；程序会自动安装并重启。更新不会覆盖本机账号密码、Excel 路径和运行数据。

## 当前项目

正式项目共九个：昆明牛、南京牛、宁波牛、长沙牛、沈阳牛、青岛白、深圳白、南京白、沈阳白。沈阳牛、沈阳白为双百度来源项目。

## 需要人工准备

- 快商通文件仍由人工导出到项目配置目录。
- 程序只使用 30 分钟内的最新导出文件；没有时按 0 对话继续，不会因为旧文件中断。
- 执行前关闭正在打开的目标 Excel/WPS 文件。

## 配置位置

- 项目与 Excel 路径：`configs/projects/*.json`
- 当前默认项目：`configs/app_config.json`
- 百度账号密码：`secrets/secrets.json`

普通同事优先使用 GUI 顶部“系统配置”，无需手工翻找文件。

## 导入授权配置

管理员完成统一账号和 API 授权后，会提供一个 `.baidu-secrets` 配置包：

1. 打开 GUI 顶部“系统配置”。
2. 点击“导入授权配置”。
3. 选择管理员提供的配置包。

程序会自动写入正确位置并立即运行项目配置检查。导入失败时会显示具体原因，可按提示重新选择文件。该操作会完整替换本机账号授权配置，但覆盖前会在后台自动保存原配置备份。

配置包是包含账号密码和 OAuth Token 的明文敏感文件，仅限公司内部使用，不得上传 GitHub、公开网盘或转发给无关人员。

## 安全说明

- 每次写 Excel 前自动备份。
- 不重建 Excel，不修改无关 sheet、公式区、汇总区和截图区。
- 程序发布包不包含本机凭据；管理员另行提供的 `.baidu-secrets` 配置包不得发给外部或上传到公开网盘、代码仓库。
- 遇到验证码、滑块或安全验证时，按界面提示人工处理。
