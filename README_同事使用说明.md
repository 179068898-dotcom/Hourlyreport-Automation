# 同事使用说明

版本：v0.4.16

百度竞价日报 / 小时报自动化工具，内部包已预置四项目配置和百度凭据。

## 开箱步骤

1. 解压到固定目录，如 `D:\自动化脚本\hourly_report_bot`
2. 双击 `run_menu.bat` 打开菜单
3. 选择 5. 文件合格校验，确认关键项通过
4. 把商务通导出 Excel/CSV 放到各项目的商务通目录

## 只需改两个路径

每个项目配置文件在 `configs/projects/项目ID.json`，一般只需改两项：

- `excel.path` — 目标 Excel 文件路径
- `kst.export_dir` — 商务通导出文件目录

其他（项目名称、账户映射、百度账号密码、日报小时报字段）已预置。

## 日常使用

双击 `run_menu.bat`，菜单顶部显示当前项目，按编号选择：

1. 小时报 → 选时段（11点/15点/18点）→ 确认 → 执行
2. 日报 → 确认 → 执行
3. 项目列表 → 选择要操作的项目
4. 项目信息 → 查看当前项目配置
5. 文件合格校验 → 检查环境是否就绪
0. 退出

运行成功后 Excel 自动打开，菜单显示完成时间。

## 注意事项

- 写入前关闭 WPS/Excel 中的目标表格
- 不要发 `secrets/secrets.json` 给外部
- 不要手动编百度数据
- JSON 路径用 `/`，不要用单个 `\`

## 双击没反应

1. 确认已解压全部文件
2. 在文件夹地址栏输入 `cmd` 回车，运行 `run_menu.bat`
3. 把黑色窗口的报错截图发回来
