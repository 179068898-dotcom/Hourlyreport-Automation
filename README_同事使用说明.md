# 同事使用说明

内部版同步日期：2026-07-22

## 最简单的使用方式

1. 双击管理员提供的 `Hourlyreport_automation_setup_v2026.7.22.107.exe`。
2. 按安装向导选择安装目录；安装完成后从桌面快捷方式启动。
3. 第一次启动会自动检查环境；缺少运行环境时会显示进度并下载隔离的项目专用 Python 3.14.5 与依赖。电脑已有的 Python 不会被使用或修改，无需卸载。
4. 在“系统”菜单中导入管理员提供的授权配置。
5. 选择项目、小时报时段或日报日期后执行；批量处理可切换“多项目”，最多选择 3 个。

GUI 成功完成小时报或日报后会自动打开项目 Excel。多项目会并行读取百度 API，再按选择顺序逐个安全写入，全部结束后统一打开成功项目的 Excel。

多项目模式不使用浏览器降级。某个项目 API 失败时会跳过并继续其他项目；点击停止时，当前项目继续完成，从下一个项目开始停止。

## 窗口与托盘

- 控制台打开时，Windows 任务栏会显示程序小图标。
- 点击控制台右上角 `X` 只会隐藏窗口，程序、托盘和正在执行的任务不会退出。
- 左键单击任务栏通知区域的托盘图标，可以重新打开控制台。
- 真正退出程序只能使用托盘图标右键菜单中的“退出程序”，或控制台“系统配置”中的“退出程序”。

## 在线更新

程序每次启动会静默检查新版本。发现新版时标题栏显示“更新”；点击后持续显示下载百分比，完成后点击“重启”。更新窗口出现后客户端才会退出，安装完成会自动重启。更新不会覆盖本机账号密码、Excel 路径和运行数据。

## 当前项目

正式项目共九个：昆明牛、南京牛、宁波牛、长沙牛、沈阳牛、青岛白、深圳白、南京白、沈阳白。沈阳牛、沈阳白为双百度来源项目。

## 需要人工准备

- 快商通文件仍由人工导出到项目配置目录。
- 程序只使用 30 分钟内的最新导出文件；没有时按 0 对话继续，不会因为旧文件中断。
- 执行前关闭正在打开的目标 Excel/WPS 文件。

## 配置位置

- 项目与 Excel 路径：`configs/projects/*.json`
- 当前默认项目：`configs/app_config.json`
- 百度账号密码：`secrets/secrets.json`

普通同事优先使用 GUI 顶部“系统”，无需手工翻找文件。

## 导入授权配置

管理员完成统一账号和 API 授权后，会提供一个 `.baidu-secrets` 配置包：

1. 打开 GUI 顶部“系统”。
2. 点击“导入授权配置”。
3. 选择管理员提供的配置包。

程序会自动写入正确位置并立即运行项目配置检查。导入失败时会显示具体原因，可按提示重新选择文件。该操作会完整替换本机账号授权配置，但覆盖前会在后台自动保存原配置备份。

## 日志记录

- 控制台实时日志使用快速逐字显示，输出较多时会自动提速。
- 全部日志会永久追加到安装目录的 `logs/gui_history.log`，切换任务或关闭窗口不会删除历史记录。

配置包是包含账号密码和 OAuth Token 的明文敏感文件，仅限公司内部使用，不得上传 GitHub、公开网盘或转发给无关人员。

## 安全说明

- 每次写 Excel 前自动备份。
- 不重建 Excel，不修改无关 sheet、公式区、汇总区和截图区。
- 程序发布包不包含本机凭据；管理员另行提供的 `.baidu-secrets` 配置包不得发给外部或上传到公开网盘、代码仓库。
- 遇到验证码、滑块或安全验证时，按界面提示人工处理。
